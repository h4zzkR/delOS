NLP Intent Parsing

This is demonstration of how NLU engine works. NLU engine is a class that provides intent classification and input message' tags extraction.

Intent - is a task that the system must do, something you want from it.

For now, you can ask system to turn light on/off in different rooms, ex:

"enable light in the garden" : turnLightOn

You can add training examples in data/nlu_engine/custom/dataset.yaml for training classifier and tagger models on them.
